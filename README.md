# 🚀 PromptForge

PromptForge is an intelligent prompt engineering tool designed to help developers and enterprises create **accurate, optimized, and cost-efficient AI prompts**. Traditional prompt optimizers often expand short prompts into long paragraphs without understanding the user’s real intent, which increases API token usage and operational costs. PromptForge solves this problem using a **human-in-the-loop agentic workflow** that analyzes the user’s input and asks targeted clarifying questions before generating the final prompt. This approach helps reduce hallucinations and improves prompt accuracy. After generating a structured master prompt, PromptForge applies **Eco-Prompting**, a token optimization technique that compresses the prompt while preserving all logical instructions and constraints, significantly reducing API token usage. The platform also includes a **Reverse Engineering module** that allows users to paste high-quality text or upload images and automatically generate the prompts required to recreate them. By combining intelligent clarification, prompt optimization, and reverse engineering, PromptForge enables developers and organizations to improve prompt quality, reduce AI costs, and gain better control over their AI workflows.

---

## ✨ Features

### 🧠 Agentic Prompt Optimization
PromptForge analyzes the user's rough prompt and asks clarifying questions such as:
- Target audience
- Tone and style
- Task objective
- Constraints and requirements

This ensures **better prompt accuracy and fewer AI hallucinations**.

### 🌱 Eco-Prompting (Token Optimization)
PromptForge compresses prompts by removing unnecessary words while preserving the meaning and constraints.

Benefits:
- Up to **60% reduction in token usage**
- Lower AI API costs
- Faster response times

### 🔄 Reverse Prompt Engineering
Users can input:
- A **piece of text**
- An **image**

PromptForge analyzes the input and generates the **prompt needed to recreate the output**, enabling quick reproduction of high-quality AI results.

---

## 🧠 How It Works

1. User enters a rough prompt idea  
2. PromptForge analyzes the input  
3. The system asks clarifying questions  
4. A structured **Master Prompt** is generated  
5. Eco-Prompting compresses the prompt  
6. The final optimized prompt is delivered
